# Contributing

Thanks for your interest in contributing to Color Hub.

## Development setup
1. Install dependencies: `pnpm install`
2. Start the dev server: `pnpm dev`
3. Load the unpacked extension from `build/chrome-mv3-dev`

## Pull requests
- Keep changes focused and small where possible.
- Update or add documentation if behavior changes.
- If you change permissions or privacy behavior, update `README.md`.

## Code style
- Prefer existing patterns and conventions in the codebase.
- Run `pnpm build` before submitting PRs when feasible.

## Reporting issues
- Provide reproduction steps and screenshots if applicable.
- Include browser version and OS.
